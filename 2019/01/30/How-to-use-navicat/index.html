如何正确食用数据库软件Navicat Premium 12

发表于 2019-01-30 | 分类于 安装新东西 | Visitors
字数统计: 351
一个非常为专门为数据库打造的软件

前言

进入官网看到这么多产品 Navicat for xxx

就在想是 Navicat for MySQL 还是 Navicat for Oracle呢

后来顿悟 当然是最牛逼的 Navicat Premium

Navicat Premium 是一套多连接数据库开发工具，让你在单一应用程序中同时连接多达七种数据库：MySQL、MariaDB、MongoDB、SQL Server、SQLite、Oracle 和 PostgreSQL，可一次快速方便地访问所有数据库。

安装操作

软件以及破解工具见 附件

安装过程一路默认就好

破解

打开注册机后

1) Patch

这里不动 均为默认选项
然后点击 patch

然后找到你的安装路径

然后出现一个这样的弹窗 别急还没彻底破解完毕

2)License, Product and Language

选择Simplified Chinese中文简体 其余不动

3)Resale License

默认 Sitelicense

4)Keygen / Offline Activation

Name 和Organization 随意填

然后点击Generate 自动生成 注册码Serial Keygen

打开Navicat Premium 12 选择注册，如果不是首次打开也可以从菜单栏的帮助选择注册

输入之前生成的注册码并点击激活

出现了一个弹窗不急
点击手动激活

将请求码复制过去然后点击Generate 将生成的东西复制回去

ok激活完毕

如果激活失败了可以参考一下这篇文章
https://blog.csdn.net/qq_39720249/article/details/80721350
